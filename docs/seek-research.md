# SEEK Portal Research Findings

## URL Structure
- Courses: `seek.onlinedegree.iitm.ac.in/courses/ns_[TERM]_[COURSECODE]`
- Assignments: `?id=53&type=assignment`
- Backend API: `backend.seek.onlinedegree.iitm.ac.in/[TERM_COURSE]/announcements`

## Term/Code Format
- `ns_22t1_cs1001` = namespace, year 2022, term 1, CS course 1001
- cs=Computer Science, hs=Humanities, ma=Mathematics, ms=Management, se=Software Engineering, ds=Data Science

## Course Codes (Foundation)
- CS1001: Computational Thinking
- CS1002: Python
- HS1001: English I
- HS1002: English II
- MA1001: Mathematics I
- MA1002: Statistics I
- MA2001: Mathematics II (for DS)
- MA2002: Statistics II

## Left Sidebar (within a course)
- Course Introduction
  - About the Course
  - Grading Policy
  - Disciplinary & Non Academic Conduct
  - Malpractice Rules
- Week 0 (some courses)
- Week 1 through Week 12
- Practice Tests (Objective)
- Mock Tests and Quizzes

## Weekly Content Types
1. **Lectures** (Video) — pre-recorded, with PPTs, slides, transcripts
2. **Activity Questions** — non-graded self-assessment
3. **Practice Assignment** — non-graded, solution PDFs available
4. **Graded Assignment** — scored, counts toward grade
5. **Tutorials** — some courses
6. **Supplementary** — SOLVE WITH US sessions, solution videos

## Question Types
- MCQ (Multiple Choice) — single correct, radio buttons, typically 2 marks
- MSQ (Multiple Select) — one or more correct, checkboxes
- SA (Short Answer/Numerical) — text/number input
- Programming — code-based

## Assignment Page Layout
- Title: "Week N | Graded Assignment N"
- Due date: YYYY-MM-DD, 23:59 IST
- Questions numbered sequentially
- Point value per question
- Submit button
- Unlimited attempts before deadline

## Grading Policy
- Weekly Graded Assignments: Best 6 of first 9 >= 40/100 for end-term eligibility
- 3 in-person invigilated quizzes per term
- 1 in-person end-term exam
- End Term Eligibility: avg weekly score >= 40/100 AND appeared for >= 1 quiz
